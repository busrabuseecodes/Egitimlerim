# 📱 Kişilik Anketi Uygulaması

Bu proje Flutter kullanılarak geliştirilmiş basit bir anket uygulamasıdır.

## ✨ Uygulama İçeriği

Uygulamada kullanıcıdan şu bilgiler alınmaktadır:

- Ad Soyad
- Cinsiyet
- Sigara kullanımı
- Günlük sigara miktarı

Girilen bilgiler ikinci sayfada “Sonucunuz” ekranında kullanıcıya gösterilmektedir.

## 🛠 Kullanılan Yapılar

- TextField
- DropdownButton
- Switch
- Slider
- ElevatedButton
- Navigator Page Transition

## 🎨 Tasarım

Uygulama mor tema kullanılarak modern bir arayüz ile tasarlanmıştır.
