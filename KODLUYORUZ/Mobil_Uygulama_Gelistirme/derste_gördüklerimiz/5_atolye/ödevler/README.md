# 📘 Dart Koleksiyonlar ve Kontrol Örnekleri

Bu proje, Dart dilinde **List, Map, Set** veri yapıları üzerinde işlem yapmayı ve koşullu ifadelerle analiz yapmayı öğretir.

---

## 📂 Dosya Yapısı

ödevler/
├── cift_sayilarin_sayisini_bulma.dart   # Listede çift sayıların sayısını bulma
├── map_teki_degerlerin_toplami.dart     # Map değerlerinin toplamını hesaplama
├── not_degerleri_analizi.dart           # Map değerlerini analiz etme
└── set_eleman_kontrolu.dart             # Set içinde eleman kontrolü


---

## 🧩 Konu Başlıkları

### 1. Çift Sayıların Sayısını Bulma
Liste üzerinde döngü ile geçilerek çift sayıların sayısı bulunur.

### 2. Map’teki Değerlerin Toplamı
Map değerleri `reduce` fonksiyonu ile toplanır ve koşullu kontrol yapılır.

### 3. Not Değerleri Analizi
Map değerleri döngü ile toplanır, toplam değere göre mesaj yazdırılır.

### 4. Set Eleman Kontrolü
`contains()` fonksiyonu ile set içinde eleman araması yapılır.

---

## ▶️ Çalıştırma
Her dosya bağımsız çalıştırılabilir. Örneğin:

```bash
dart run koleksiyon_ornekleri/cift_sayilarin_sayisini_bulma.dart
```
